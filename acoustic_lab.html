String Theory

A stretched string under tension supports transverse waves. For a string fixed at both ends, standing waves form when the length equals integer multiples of half-wavelengths.

Fundamental Equation

f₁ = (1/2L) · √(T/μ)

where T = tension (N), L = length (m), μ = linear mass density (kg/m)

Linear Mass Density

For a solid cylindrical string:
μ = ρ · π · (d/2)²

where ρ = material density, d = diameter

Wound Strings

Wound strings consist of a core wire wrapped with a winding wire. The total linear density is the sum of core and winding contributions. This allows lower frequencies at practical tensions without excessive diameter.

Inharmonicity

Real strings have stiffness, causing overtones to deviate from exact integer multiples. Thicker, stiffer strings exhibit more inharmonicity, which affects timbre and tuning.

Organ Pipe Theory

An organ pipe is a resonant air column. Sound is produced by an air jet interacting with a sharp edge (labium), setting the air column into vibration.

Open Pipes

Both ends are pressure nodes (velocity antinodes). All harmonics are present.
fₙ = n · c / 2Lₐ   for n = 1, 2, 3, ...

Closed (Stopped) Pipes

One end is a pressure antinode (closed), the other a node (open). Only odd harmonics.
fₙ = n · c / 4Lₐ   for n = 1, 3, 5, ...

End Correction

The acoustic length Lₐ exceeds the physical length by an end correction ΔL:
Lₐ = L + ΔL

For a circular open end:
ΔL ≈ 0.6 · r = 0.3 · d

where r = pipe radius, d = diameter. The factor varies with geometry and frequency.

Conical Pipes

Conical pipes support all harmonics like open cylindrical pipes but with different impedance characteristics. The taper affects radiation efficiency and timbre.

Speed of Sound in Air

c = 331.3 · √(1 + T/273.15) m/s
where T = temperature in °C. At 20°C, c ≈ 343.2 m/s.
